Bedienungsanleitung
Unfallaufnahme v3.1 · PI Telfs
Die App fuehrt durch die strukturierte Aufnahme eines Verkehrsunfalls in 5 Schritten. Alle Daten werden lokal gespeichert.
📍 Schritt 1 - Unfallort
• GPS-Taste antippen → Strasse + Gemeinde automatisch
• Google Maps Link erscheint automatisch
• Gemeinde: Freitext mit Tirol-Vorschlaegen
• Wetter, Strassenzustand, Licht auswaehlen
• Bei Verletzten: Rettungskraefte auswaehlen
🚗 Schritt 2 - Fahrzeuge
• Marke + Farbe aus Dropdown waehlen
• Foto → Kennzeichen automatisch (OCR, ca. 15 Sek.)
• Zulassungsschein direkt beim Fahrzeug fotografieren
• Beifahrer koennen beim Fahrzeug eingetragen werden
👤 Schritt 3 - Personen
• Rolle: Lenker, Beifahrer, Fussgaenger, Zeuge
• Fuehrerschein Foto → Name + Geb.datum automatisch
• Verletzungsgrad eintragen (keine / leicht / schwer)
• Versicherungsdaten eintragen
📷 Schritt 4 - Fotos
• Gesamtansicht der Unfallstelle
• Spurenlage auf der Fahrbahn
• Strassenschilder und Kilometerstein
• Beliebig viele Fotos hinzufuegen
📤 Schritt 5 - Export
• Cryptshare: Sicher via cryptshare.bmi.gv.at (nur Polizei-Netz)
   1. Absender-Mail eintragen → OTP anfordern
   2. OTP aus Mail eintragen → Hochladen
• ZIP lokal: PDF + alle Fotos als ZIP speichern
💡 Tipps
• Speichern: Disk-Icon oben rechts → Projekt wird gespeichert
• Mehrere Projekte: Zurueck-Pfeil → Projektliste
• OCR: Gute Beleuchtung + gerade Kamera = bessere Erkennung
• App als Vollbild: Browser → "Zum Startbildschirm hinzufuegen"
